⚙ How to add a new tool to this browser ▼ Open this HTML file in any text editor (Notepad, VS Code, etc.) and find the line that reads const T=[{"n": "VM Ware Cloud Director"....

Add a new entry inside that array using the same JSON format as the other tools. Each tool has these fields: n (name), c (category/dept), s (subcategory), t (tags), web (Web-Based / Installed / Both), desc (description), team (array of {name, access, training}), poc, email, phone, url, cpu (cost per user/month), mco (monthly cost overall), int (integrations), use (how we use it), pct, cdt, exp (expiration), ctp (contract type).

After editing: Save the file and re-upload to SharePoint to replace the old version. That's it!
